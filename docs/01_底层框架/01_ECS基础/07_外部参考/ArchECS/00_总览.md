# ArchECS 总览

# 1 本目录定位
## 1.1 读者与范围
面向需要理解 Arch ECS 官方能力边界与在 Ludots 中如何吸收的读者。本目录包含“外部参考正文（对 Ludots 的吸收点）”与“官方文档摘录（证据材料）”。
## 1.2 本目录边界
-
  - 本目录不作为项目口径真源：项目口径必须回填到 `docs/01_底层框架/01_ECS基础/07_使用手册/*` 与相关子系统 SSOT。
  - 本目录不尝试完整镜像官方 GitBook，仅保留与 Ludots 直接相关的关键页面节选。
## 1.3 推荐阅读顺序
-
  - 先读结论：`01_Arch_ECS_外部参考.md`
  - 再看证据：Why Arch → World → Archetypes&Chunks → Query → CommandBuffer → Jobs/Scheduler → Events

# 2 SSOT
## 2.1 代码入口
- `src/Libraries/Arch/`
- `src/Libraries/Arch.Extended/`
## 2.2 口径说明
-
  - 本目录引用的官方口径以页面摘录为准；与仓库内 vendored 源码不一致时，Ludots 以 vendored 源码为准并在外部参考中显式记录差异与风险。
## 2.3 关键约束
-
  - 默认不做向后兼容与静默 fallback：口径冲突必须显式裁决，运行时保持单一真源与 fail-fast。
  - 结构变更必须集中到安全阶段执行，不允许在 Query/并行 Job 中直接变更 World。
  - 热组件 0GC：禁止托管引用飞线。
