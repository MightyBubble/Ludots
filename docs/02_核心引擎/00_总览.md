# 核心引擎 总览

# 1 本目录定位

本目录承载引擎生命周期、系统分组与执行时序、时钟与节拍器、TimeSlice、配置管线等核心引擎能力。

## 1.1 读者与范围

读者：

- 引擎/基建开发者：需要理解引擎 SSOT 与跨模块边界口径。
- 上层系统开发者：需要知道预算/确定性/失败策略的硬约束与接入方式。

## 1.2 本目录边界

- 只承载引擎侧基础设施与口径裁决，不承载业务语义。  
- 外部参考不在本层沉淀，必须归档到 `docs/07_附录与扩展/external/`。  

## 1.3 推荐阅读顺序

1. `docs/02_核心引擎/01_总架构/00_总览.md`
2. `docs/02_核心引擎/02_系统分组与执行时序/00_总览.md`

# 2 SSOT

## 2.1 代码入口

- 引擎生命周期与装配：`src/Core/Engine/GameEngine.cs`
- 时钟：`src/Core/Engine/ClockFoundation.cs`
- Pacemaker：`src/Core/Engine/Pacemaker/`

## 2.2 口径说明

- Phase 顺序与暂停/恢复以 cooperative simulation 的实现为准：`src/Core/Engine/Pacemaker/PhaseOrderedCooperativeSimulation.cs`。  

## 2.3 关键约束

1. 确定性：排序/tie-break 必须固化，可回放。  
2. 可预算：长任务必须 time-slice，可熔断，可恢复。  
3. fail-fast：禁止静默 fallback，缺失依赖必须可观测并中止。
