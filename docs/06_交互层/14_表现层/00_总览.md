# 表现层 总览

# 1 本目录定位

本目录定义逻辑到表现的同步与呈现：渲染命令缓冲、相机系统、可见性裁剪、HUD、坐标映射器，以及 VisualSync 系统。
表现层只读玩法真源，允许丢帧/降级但必须可观测；不允许反向影响玩法裁决。

# 2 SSOT（代码入口）

- `src/Core/Presentation/`
- `src/Core/Systems/`（可见性/同步相关系统）
- `src/Core/Gameplay/Camera/`
