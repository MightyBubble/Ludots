# 输入系统 总览

# 1 本目录定位

本目录定义输入系统：输入配置（bindings、context/action）、运行时输入处理（PlayerInputHandler/Context 栈）、以及输入后端抽象（IInputBackend）。
输入系统只产生“意图”，不得写入玩法真源；玩法真源由系统在固定相位内消费意图并推进。
