# 总览入口 标题与必选章节

# 1 标题规范

一级标题固定：

- `# {目录名} 总览`

# 2 必选章节

- `# 1 本目录定位`
  - `## 1.1 读者与范围`
  - `## 1.2 本目录边界`
  - `## 1.3 推荐阅读顺序`
- `# 2 SSOT`
  - `## 2.1 代码入口`
  - `## 2.2 口径说明`
  - `## 2.3 关键约束`
- `# 3 子文档索引`
  - `## 3.1 子文档清单`
  - `## 3.2 文档类型与阅读路径`

裁决：

- “SSOT”必须至少包含 1 条仓库相对路径的代码入口，并说明它为何是口径真源。
- “子文档索引”必须覆盖本目录所有子文档，并保证引用稳定（不依赖作者习惯与临时链接）。
- 若本目录存在 `99_KANBAN.md`，必须在“子文档清单”中明确标注其为开发计划唯一真相。

# 3 可选章节

按需选用。

- `# 4 阅读路径建议`
- `# 5 与其他目录的关系`
- `# 6 常见问题`
