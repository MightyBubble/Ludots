# 文档总览 总览

# 1 本目录定位

## 1.1 读者与范围

本目录承载 `docs/` 的唯一入口与全局规则：阅读路径、SSOT 清单、文档类型与骨架、迁移映射表。

## 1.2 本目录边界

- 本目录只定义文档体系与写作规范，不替代任何子系统的 SSOT。  
- 本目录所有引用必须使用仓库相对路径，禁止本机绝对路径与本地 file scheme 链接。  

## 1.3 推荐阅读顺序

1. `docs/00_README.md`
2. `docs/00_文档总览/01_文档规范.md`
3. `docs/00_文档总览/规范/01_子文档路径与命名规范.md`

# 2 SSOT

## 2.1 代码入口

本目录的 SSOT 是文档规范与模板集合：

- `docs/00_文档总览/01_文档规范.md`
- `docs/00_文档总览/规范/`

## 2.2 口径说明

- 目录与命名的裁决以 `docs/00_文档总览/规范/01_子文档路径与命名规范.md` 为准。  
- 各文档类型的必选章节以 `docs/00_文档总览/规范/10_总览入口.md` 等模板为准。  

## 2.3 关键约束

1. 每个子系统目录必须有 `00_总览.md` 作为唯一阅读入口。  
2. 一旦选择布局 B（按类型分组），不得再向目录根部新增散文档。  
3. 旧路径需要保留时只能用“兼容入口”，正文只写新位置，不复述内容。
