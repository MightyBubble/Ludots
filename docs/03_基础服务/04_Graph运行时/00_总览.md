# Graph 运行时 总览

# 1 本目录定位

本目录定义领域无关的 GraphRuntime：GraphProgram（Blob/指令流）、GraphExecutor（确定性执行、指令预算熔断）、以及节点库与 RuntimeAPI 注入的分层。
GraphRuntime 不绑定任何领域语义（GAS/AI/UI 等只能以节点库形式接入）。

## 1.1 读者与范围

读者：

- 基建开发者：实现 GraphProgram 编译/加载与执行器的确定性与预算口径。
- 上层系统开发者：以"节点库"形式接入领域逻辑，而不是把 Graph 当脚本引擎滥用。

范围：

- GraphProgram：指令流/常量池/元信息（可序列化 blob）
- GraphExecutor：确定性执行、指令预算熔断、fail-fast 未注册 opcode
- 节点库：以模块化方式把领域能力接到 GraphRuntime（当前已实现 GASGraph）

## 1.2 本目录边界

- GraphRuntime 不定义领域语义（GAS/AI/UI）；领域逻辑只能以节点库接入。
- TimeSlice 预算语义不在本目录裁决，引用核心引擎 SSOT。

## 1.3 推荐阅读顺序

1. `docs/03_基础服务/04_Graph运行时/01_GraphExecutor.md` — 执行器接口规范
2. `docs/03_基础服务/04_Graph运行时/02_GraphProgram_接口规范.md` — 程序表示规范
3. `docs/03_基础服务/04_Graph运行时/03_NodeLibraries_架构设计.md` — 节点库分层架构
4. `docs/03_基础服务/04_Graph运行时/04_Graph去重重构_对齐报告.md` — 审计与修复记录
