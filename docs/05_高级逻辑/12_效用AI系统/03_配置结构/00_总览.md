# 配置结构 总览

# 1 本目录定位
## 1.1 读者与范围

本目录定义效用AI系统可数据驱动部分的配置结构、加载期强校验与版本策略，服务 UGC 与 Mod/VFS 的内容注入。

## 1.2 本目录边界

- 不定义业务语义；配置只能使用 token/ID 引用（禁止 magic string）。
- 不承担运行时代码实现细节；仅描述消费点与校验规则。

## 1.3 推荐阅读顺序

1. `03_响应曲线_配置结构.md`
