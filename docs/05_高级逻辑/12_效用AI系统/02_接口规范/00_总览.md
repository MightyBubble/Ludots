# 接口规范 总览

# 1 本目录定位
## 1.1 读者与范围

本目录定义效用AI系统对外/对内依赖的接口契约：索敌查询（TargetingQuery）与空间查询适配层（SpatialQuery）。

## 1.2 本目录边界

- 仅描述接口形状、约束与失败策略；不承载实现细节。
- 不引入业务语义；语义只通过 token/ID 注入。

## 1.3 推荐阅读顺序

1. `01_索敌查询.md`
2. `02_空间查询适配层.md`

# 2 SSOT
## 2.1 代码入口

- `src/Core/Gameplay/GAS/Orders/OrderQueue.cs`
- `docs/03_基础服务/06_空间服务/03_空间查询接口.md`

## 2.2 口径说明

- SpatialQuery 负责产出候选快照（固定上界），TargetingQuery 只读消费并产出 Order（真源）。

## 2.3 关键约束

- 候选上限 16、确定性、零 GC、离散 Clock 驱动、fail-fast。
